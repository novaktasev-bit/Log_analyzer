# Simple Log Analyzer

A Python-based utility designed for QA testing and log monitoring. This tool processes system messages, identifies errors, and provides a stability report.

## Key Features
- Case-insensitive Error Detection: Automatically finds "error" regardless of capitalization.
- Stability Statistics: Calculates the error rate percentage with two-decimal precision.
- Detailed Reporting: Stores and displays all identified error messages for debugging purposes.

## How it Works
1. Run the script.
2. Enter your log entries when prompted.
3. Type 'stop' to generate the final summary and error list.

## Technical Details
- Language: Python 3.x
- Core Concepts: Functions, While and For loops, Lists, String manipulation.
